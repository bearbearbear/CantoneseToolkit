# 普通话口语→粤语口语离线翻译数据 v0.1.0

构建日期：2026-07-19

## 数据规模

- 固定短语：396 条
- 词汇映射：331 条
- 句型模板：91 条
- 语法与后处理规则：78 条
- 测试用例：566 条
- 翻译引擎运行单元合计：896 条

## 定位

本数据包面向完全离线的网页翻译器，输入为普通话口语化简体文本，输出为现代香港粤语口语书面形式（繁体粤文）。
首版重点覆盖基础交流、餐饮、购物、交通、酒店、社交、工作和医疗场景。

## 文件

- `phrases.csv/jsonl`：完整句和高频固定短语，优先级最高。
- `lexicon.csv/jsonl`：普通话词语到粤语词语的条件化映射。
- `templates.csv/jsonl`：带槽位的句型模板。
- `rules.csv/jsonl`：否定、疑问、体貌、比较、语序、语气等改写规则。
- `tests.csv/jsonl`：精确短语、模板、规则和组合测试。
- `runtime-core-v0.1.0.json`：浏览器 Worker 可直接加载的原型运行包。
- `schema.json`：字段定义。
- `metadata.json`：版本、数量、来源说明和文件哈希。

## 建议运行优先级

1. 输入规范化和动态槽位保护
2. 完整短语匹配
3. 最长短语匹配
4. 句型模板匹配
5. 按 `stage` 和 `priority` 执行规则
6. 词汇映射
7. 候选评分
8. 语气词和标点清理

推荐优先级：完整句 > 长短语 > 模板 > 规则 > 单词。

## 模板槽位

- `NP`：名词短语
- `VP`：动词短语
- `PERSON`：人物
- `PLACE`：地点
- `TIME`：时间
- `NUMBER`：数字或数量
- `ITEM`：商品、菜品或物品
- `ADJ`：形容词
- `CLAUSE`：短分句

## 审校要求

所有记录当前均标记为 `draft_curated`，建议在产品使用前完成：

1. 香港粤语母语编辑逐条审校；
2. 对 `condition` 和规则冲突进行引擎测试；
3. 用真实用户输入扩充测试集；
4. 将存在地区差异的表达增加 `variant` 和候选排序；
5. 对中英夹杂表达提供开关。

## 已知限制

- 暂不支持长文本、复杂嵌套从句和跨句上下文。
- 暂不包含粤拼、读音和音频。
- 有些规则需要词性、语义和场景条件，不能直接当作无条件字符串替换。
- 测试集主要验证目标输出，不代表覆盖所有可接受粤语变体。
